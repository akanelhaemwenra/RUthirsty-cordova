## 🎨 设计理念

### 视觉层次
1. **背景层** - 动态图案
2. **卡片层** - 渐变+阴影
3. **内容层** - 文字+图标
4. **装饰层** - 光效+边框

### 色彩系统
- **主色**: #667eea (紫蓝色)
- **辅色**: #764ba2 (深紫色)
- **强调色**: #ff4757 (红色)
- **背景**: 渐变紫色
- **文字**: #333 / #666 / #999

### 动画原则
- **自然**: 模拟物理运动
- **流畅**: 60fps 性能
- **有意义**: 提供反馈
- **不过度**: 适度使用
