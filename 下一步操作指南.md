## 💡 常见问题

### Q1: 天气显示"天气信息不可用"
**A:** 这是正常的，因为还没有配置 OpenWeatherMap API Key。

**解决方案：**
1. 访问 https://openweathermap.org/api
2. 注册并获取免费 API Key
3. 在 www/js/index.js 中配置
4. 刷新浏览器

### Q2: 日历显示不正确
**A:** 检查设备系统时间设置是否正确。

### Q3: 地理定位失败
**A:** 这是正常的，应用会自动使用默认城市（北京）。

**原因：**
- 用户拒绝了位置权限
- 浏览器不支持地理定位
- 定位超时

### Q4: 如何清除天气缓存
**A:** 在浏览器控制台执行：
```javascript
localStorage.removeItem('weatherCache')
localStorage.removeItem('weatherCacheTime')
```

### Q5: 构建 APK 失败
**A:** 参考 cordova-apk-builder 文档：
```bash
cat cordova-apk-builder/references/troubleshooting.md
```
