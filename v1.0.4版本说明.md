# v1.0.4 版本说明

## 🎉 新功能

### 📅 日历显示
- 实时显示当前日期（年月日）
- 显示当前星期
- 每分钟自动更新
- 美观的卡片式设计

### 🌤️ 天气显示
- 实时显示当前温度
- 显示天气状况（晴、多云、雨等）
- 天气图标（emoji）
- 基于地理位置的智能定位
- 30分钟智能缓存，减少API调用

### 💡 智能天气提醒
根据天气情况自动调整喝水提醒：
- 炎热天气（>30°C）：提醒多补充水分
- 温暖天气（>25°C）：提醒保持水分
-寒冷天气（<10°C）：建议喝温水
- 下雨天：个性化提醒
- 晴天：鼓励保持活力

## 🚀 使用说明

### 配置天气功能

1. **获取 OpenWeatherMap API Key**
   - 访问：https://openweathermap.org/api
   - 注册账号并获取免费 API key
   - 免费限额：60次/分钟，1,000,000次/月

2. **配置 API Key**
   - 打开 `www/js/index.js`
   - 找到两处 `YOUR_API_KEY_HERE`
   - 替换为你的 API key

```javascript
// 第 548 行附近
var apiKey = 'YOUR_API_KEY_HERE'; // 替换为你的 key

// 第 556 行附近
var apiKey = 'YOUR_API_KEY_HERE'; // 替换为你的 key
```

3. **重新构建应用**
```bash
npx cordova build android
```

### 不配置 API Key 的影响
- 天气显示为 "天气信息不可用"
- 其他功能正常使用
- 不影响日历显示

## 🐛 已知问题

### 地理定位
- 首次使用需要授权位置权限
- 某些设备可能定位失败（会自动使用默认城市）
- 定位超时设置为10秒

### 天气API
- 需要网络连接
- 免费账户有请求限制
- 缓存可能导致数据延迟（最多30分钟）
