### 🐛 已知问题

#### 地理定位
- 首次使用需要用户授权位置权限
- 某些设备可能定位失败（自动使用北京）
- 定位超时设置为10秒
- 浏览器环境可能不支持

#### 天气API
- 需要配置OpenWeatherMap API Key
- 需要网络连接
- 免费账户限制：60次/分钟，1,000,000次/月
- 缓存可能导致数据延迟（最多30分钟）
- 未配置API Key时显示"天气信息不可用"

### 📝 使用说明

#### 配置天气功能

**步骤1：获取API Key**
1. 访问 https://openweathermap.org/api
2. 注册免费账号
3. 获取API Key（免费限额充足）

**步骤2：配置到应用**
1. 打开 `www/js/index.js`
2. 找到第548行和556行的 `YOUR_API_KEY_HERE`
3. 替换为你的API Key

```javascript
// 第 548 行附近
var apiKey = 'YOUR_API_KEY_HERE'; // 替换为：'abc123...'

// 第 556 行附近
var apiKey = 'YOUR_API_KEY_HERE'; // 替换为：'abc123...'
```

**步骤3：重新构建**
```bash
npx cordova build android
```

#### 不配置API Key的影响
- ✅ 日历功能正常使用
- ⚠️ 天气显示"天气信息不可用"
- ✅ 其他功能不受影响
